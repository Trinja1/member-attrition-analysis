# Member Attrition Analysis — Excel Dashboard

## Project Overview
End-to-end member attrition analysis for a subscription-based
business built in Microsoft Excel.

## What's Included
- 100-member simulated dataset with usage history and plan data
- KPI dashboard with dynamic Excel formulas and status indicators
- Plan-level attrition breakdown by subscription tier
- 18-month trend analysis with line chart visualization
- Written insights and recommendations section

## Skills Demonstrated
Excel · COUNTIF · COUNTIFS · AVERAGEIF · IF · Dashboard Design ·
Attrition Analysis · Business Performance Reporting
